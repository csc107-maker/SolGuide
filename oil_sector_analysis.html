📊 유가 급등 업종 피해 vs 수혜 완전 분석

원유 재고 급감·고유가 국면에서의 업종별 영향도 분석 (2026.05 기준)

📉 직격탄 피해 업종
✈️ 항공·여행
항공유 원가 폭등, 노선 축소, 여행 수요 위축. 가장 민감한 섹터.
KAL / 아시아나 / 델타(DAL)
🚢 해운·물류
운항 원가 급증. 호르무즈 통항 불능으로 우회항로 비용 폭발.
HMM / 팬오션 / 머스크(MAERSK)
🧪 석유화학·정유
나프타 등 원료 조달 위기. 중동 의존도 70% 이상으로 원가 압력 극심.
LG화학 / 롯데케미칼 / 한화솔루션
🏭 제조업 전반
에너지 비용·원자재 가격 동반 상승으로 영업이익률 하락.
현대차 / 삼성전자 / POSCO
💊 제약·바이오
인플레 → 금리인하 기대 약화 → 밸류에이션 하락 압력.
셀트리온 / 삼성바이오 / 에이치엘비
🛒 소비재·유통
물류비·에너지비 상승 → 마진 압박. 소비자 실질 구매력 감소.
이마트 / 쿠팡(CPNG)
📈 반사 수혜 업종
⛽ 정유·에너지 메이저
유가 상승 = 직접 이익 증가. 탐사·생산(E&P) 현금흐름 폭발적 개선.
에쓰오일(S-OIL) / GS칼텍스 / XOM / CVX
🛡️ 방산·항공우주
지정학 리스크 고조 = 방위비 증가 수혜. 미사일·드론 방어 시스템 수요 급증.
한화에어로스페이스 / LIG넥스원 / ITA ETF
🥇 금·원자재
안전자산 수요 급증. 인플레 헤지 목적 금 매수세 가속화.
GLD ETF / IAU / 고려아연
☀️ 재생에너지·신재생
고유가 → 대체에너지 수요 부각. 태양광·풍력·원전 투자 확대.
OCI홀딩스 / 씨에스윈드 / 두산에너빌리티
🔋 배터리·전기차
고유가 지속 → 내연기관 대비 전기차 경제성 부각.
LG에너지솔루션 / 에코프로비엠 / TSLA
🏦 금융·배당주
고금리 지속 환경에서 배당 안정성 높은 종목 선호 심리 강화.
KB금융 / 신한지주 / 메리츠금융지주
